# **omi**

_جهاز ارتداء بالذكاء الاصطناعي لاجتماعاتك ومحادثاتك الفورية: التقاط، تلخيص، واتخاذ إجراءات—يدكم حرة._

تعرف على Omi، وهو أول جهاز ارتداء مفتوح المصدر للذكاء الاصطناعي في العالم يلتقط المحادثات، ويُنتج الملخصات وعناصر العمل، ويُنفّذ المهام نيابة عنك. ما عليك سوى توصيل Omi بهاتفك المحمول لتستمتع بتفريغ فوري عالي الجودة للاجتماعات والدردشات والمذكّرات الصوتية في أي مكان.

### في هذا المستودع

ملاحظة: قد تذكر المراجع القديمة `personas-open-source` في جذر المستودع؛ وفي هذا المستودع هو حاليًا تحت `web/personas-open-source`.

### مخطط عام

```text
.
├── app/                     # تطبيق Flutter المرافق
├── backend/                 # FastAPI backend
├── docs/                    # توثيق Mintlify
├── mcp/                     # خادم MCP للذكريات/المحادثات
├── omi/                     # Omi hardware + firmware
├── omiGlass/                # Omi Glass hardware + firmware + app
├── plugins/                 # Integrations/apps/plugins
├── sdks/                    # python / react-native / swift
├── web/frontend/            # الواجهة الأمامية الأساسية لـ Next.js
├── web/personas-open-source/ # تطبيق ويب Personas
├── scripts/                 # أدوات مساعدة (format hooks, OTA helper)
└── i18n/                    # محتوى متعدد اللغات وترجمات
```

## <a id="installation"></a>التثبيت

بما أن هذا مستودع مونوربو، نفّذ التثبيت لكل مكوّن على حدة.

### استنساخ المستودع

```bash
git clone https://github.com/BasedHardware/omi.git
cd omi
```

### App (Flutter)

```bash
cd app
bash setup.sh ios
# أو
bash setup.sh android

flutter run --flavor dev
```

### Backend (FastAPI)

```bash
cd backend
cp .env.template .env
pip install -r requirements.txt
uvicorn main:app --reload --env-file .env
```

### الواجهة الأمامية الأساسية (Next.js)

```bash
cd web/frontend
npm install
cp .env.template .env.local
npm run dev
```

### تطبيق web personas (Next.js)

```bash
cd web/personas-open-source
npm install
npm run dev
```

### MCP server

```bash
cd mcp
# راجع mcp/README.md لإعداد Claude Desktop المعتمد على Docker
```

### SDKs

```bash
cd sdks/python && pip install -e .
cd ../react-native && npm install
# حزمة Swift متاحة عبر Package.swift في الجذر كـ `omi-lib`
```

## <a id="usage"></a>الاستخدام

### مسار التطبيق النهائي

1. ثبّت تطبيق Omi من روابط المتجر أعلاه.
2. اربط/استخدم جهاز Omi.
3. أنشئ تطبيقًا عبر مسار البدء السريع لـ webhook.
4. تكلّم وتحقق من أحداث التفريغ النصي المباشر في endpoint webhook الخاص بك.

### backend محلي مع التطبيق

1. شغّل backend محليًا على المنفذ `8000`.
2. اعرض backend محليًا عبر نطاق ثابت في ngrok.
3. عيّن `BASE_API_URL` في التطبيق على رابط ngrok.
4. استخدم تطبيق Omi مقابل backend المحلي.

### سيرات عمل SDK

- Python SDK: اكتشف أجهزة BLE عبر `omi-scan`، فك ترميز حزم Opus، وقم بالتفريغ النصي عبر Deepgram.
- React Native SDK: الاتصال، بث بايتات الصوت، فحص codec/البطارية.
- Swift SDK (`omi-lib`): scan/connect/transcribe باستخدام واجهات `OmiManager` APIs.

## <a id="configuration"></a>الإعدادات

الإعدادات موزعة حسب المكوّن. استخدم قالب/README الخاص بكل مكوّن كمصدر مرجعي.

- `backend/.env.template` -> مفاتيح API ونفاذ البنية الأساسية في backend
- `web/frontend/.env.template` -> متغيرات بيئة الواجهة الأمامية للويب
- قيم بيئة `app` -> إعدادات تشغيل تطبيق الجوال
- متغيرات mcp -> `OMI_API_KEY` و`OMI_API_BASE_URL` الاختياري
- مفاتيح خاصة بـ SDK:
  - Python SDK يستخدم `DEEPGRAM_API_KEY`
  - Personas يستخدم متغيرات Firebase/OpenRouter/RapidAPI/Mixpanel

إذا لم تكن متأكدًا من المتغيرات المطلوبة، احتفظ بالقيم الافتراضية واتبع README المكوّن المحدد قبل حذف/تعديل القيم.

## <a id="troubleshooting"></a>استكشاف الأخطاء

- مشكلة نموذج/شبكة في إعداد محلي backend: إذا واجهت مشكلة التنزيل المعروفة للـ SSL، طبّق حل `ssl` الموثّق في `backend/README.md` وأعد المحاولة.
- لا يمكن للتطبيق الوصول إلى backend المحلي: تحقق من تشغيل نطاق ngrok وأن `BASE_API_URL` يشير بالضبط إلى هذا الرابط العام.
- مشاكل اكتشاف BLE: تأكد من منح أذونات Bluetooth لمنصة التشغيل (Terminal/Xcode/تطبيق Android حسب الحالة).
- عدم تطابق المسارات في المستندات القديمة: قد تظهر مراجع قديمة مسارات غير محدثة (مثل `personas-open-source` في الجذر أو فرق حالة `omi/OmiGlass`). استخدم مخطط الأدلة الحالي الظاهر أعلاه.
