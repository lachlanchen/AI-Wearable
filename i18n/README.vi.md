# **omi**

_AI-powered wearable for instant meetings and chats: capture, summarize, and act—hands-free._

Thiết bị đeo AI cho cuộc họp và trò chuyện tức thời: ghi lại, tóm tắt và thực thi—không cần dùng tay.

Khám phá Omi, thiết bị đeo AI mã nguồn mở hàng đầu thế giới giúp ghi lại hội thoại, tạo bản tóm tắt, liệt kê việc cần làm và thực hiện các thao tác cho bạn. Chỉ cần kết nối Omi với thiết bị di động, bạn sẽ có transcript chất lượng cao tự động cho các cuộc họp, chat và ghi chú giọng nói mọi lúc, mọi nơi.

## Tổng quan

Kho mã này là monorepo mã nguồn mở của hệ sinh thái Omi.

| Bao gồm | Chi tiết |
| --- | --- |
| 🧠 Omi Hardware | Firmware thiết bị đeo Omi + tài nguyên phần cứng |
| 🥽 Omi Glass | Firmware/phần cứng/app của Omi Glass |
| 📱 Di động | Ứng dụng đồng hành Flutter |
| 🌐 Backend | Dịch vụ FastAPI |
| 🕸️ Web | Frontend chính của Omi + Personas |
| 🧩 Hệ sinh thái | Ecosystem plugin/tích hợp |
| 📦 SDKs | Python, React Native, Swift |
| 🔗 Hạ tầng Agent | Máy chủ MCP |
| 📘 Docs | Nguồn tài liệu |

README gốc tập trung vào khởi tạo và quy trình làm việc hàng ngày của nhà phát triển. Hướng dẫn thiết lập theo từng thành phần và các ghi chú vận hành sâu hơn nằm trong README của từng subproject và trong `docs/`.

### Giả định khi thiết lập

- Một số lệnh phụ thuộc vào credential bên ngoài (Firebase, OpenAI/Deepgram/các nhà cung cấp API), thường được cấu hình theo từng thành phần qua template `.env` của component đó. Nếu thiếu giá trị trong tài liệu cục bộ, hãy xem README thành phần tương ứng.
- Nếu bạn chỉ cần một bề mặt, chỉ cài/build phần đó để giữ nhịp iteration nhanh.

## Cài đặt

Vì đây là monorepo, hãy cài đặt theo từng component.

### Clone repository

```bash
git clone https://github.com/BasedHardware/omi.git
cd omi
```

### App (Flutter)

```bash
cd app
bash setup.sh ios
# or
bash setup.sh android

flutter run --flavor dev
```

### Backend (FastAPI)

```bash
cd backend
cp .env.template .env
pip install -r requirements.txt
uvicorn main:app --reload --env-file .env
```

### Frontend chính (Next.js)

```bash
cd web/frontend
npm install
cp .env.template .env.local
npm run dev
```

### Personas web app (Next.js)

```bash
cd web/personas-open-source
npm install
npm run dev
```

### MCP server

```bash
cd mcp
# See mcp/README.md for Docker-based Claude Desktop setup
```

### SDKs

```bash
cd sdks/python && pip install -e .
cd ../react-native && npm install
# Swift package is exposed via root Package.swift as `omi-lib`
```

## Cách sử dụng

### Luồng app người dùng cuối

1. Cài ứng dụng Omi từ các liên kết store ở trên.
2. Kết nối và sử dụng thiết bị Omi.
3. Tạo app qua luồng bắt đầu nhanh webhook.
4. Nói chuyện và theo dõi sự kiện transcript thời gian thực tại webhook endpoint của bạn.

### Backend local với app

1. Khởi động backend cục bộ ở cổng `8000`.
2. Mở backend local bằng miền tĩnh ngrok.
3. Đặt `BASE_API_URL` trong app thành URL ngrok của bạn.
4. Dùng ứng dụng Omi với backend local.

### Luồng SDK

- Python SDK: quét thiết bị BLE bằng `omi-scan`, giải mã gói Opus, và chép lời qua Deepgram.
- React Native SDK: kết nối, truyền audio bytes, kiểm tra codec và pin.
- Swift SDK (`omi-lib`): quét/kết nối/chép lời bằng API `OmiManager`.

## Cấu hình

Cấu hình được phân phối theo component. Dùng template/README của từng subproject làm nguồn tham chiếu.

- `backend/.env.template` -> khóa API/cấu hình hạ tầng backend
- `web/frontend/.env.template` -> biến môi trường frontend web
- giá trị môi trường `app` -> cấu hình runtime ứng dụng di động
- biến môi trường `mcp` -> `OMI_API_KEY` và `OMI_API_BASE_URL` tùy chọn
- Khóa riêng cho từng SDK:
  - Python SDK dùng `DEEPGRAM_API_KEY`
  - Personas dùng biến môi trường Firebase/OpenRouter/RapidAPI/Mixpanel

Nếu chưa chắc biến nào là bắt buộc, hãy giữ mặc định và theo README thành phần đó trước khi xoá/thay đổi giá trị.

## Ghi chú phát triển

- CI/CD và tự động hóa triển khai được cấu hình tại:
  - `.github/workflows/`
  - `codemagic.yaml`
- `scripts/pre-commit` ở root định dạng các file đã staged cho:
  - Dart trong `app/`
  - Python trong `backend/`
  - C/C++ trong `omi/` và `omiGlass/`
- `Package.swift` tại root publish package Swift `omi-lib` từ `sdks/swift`.
- Thư mục `i18n/` tồn tại để chứa README/nội dung đa ngôn ngữ.

## Khắc phục sự cố

- Lỗi model/mạng khi chạy local backend: nếu gặp vấn đề SSL tải xuống đã biết, áp dụng workaround `ssl` trong `backend/README.md` rồi thử lại.
- App không kết nối được backend local: xác nhận domain ngrok đang chạy và `BASE_API_URL` trỏ đúng URL public đó.
- Vấn đề phát hiện BLE: xác nhận quyền Bluetooth đã được cấp (Terminal/Xcode/app Android tùy theo nền tảng).
- Mismatch đường dẫn trong tài liệu cũ: một số tài liệu tham chiếu di sản có thể vẫn dùng đường dẫn lỗi thời (ví dụ `personas-open-source` ở root hoặc phân biệt chữ hoa/thường `omi/OmiGlass`). Trong repository này dùng đúng layout thư mục hiện tại như ở trên.
